How the analysis works
① Search — Fetches up to N case headnotes (short summaries) matching your keywords. These come free with every search result — no extra downloading.
② Pre-screen — All N headnotes are sent to the AI in a single call. It reads them all at once and picks the ~20 most relevant case IDs. Fast: one LLM call regardless of pool size.
③ Deep score — Only for those ~20 cases: the full case text is fetched (from cache if available, otherwise downloaded), then scored 0–100 across: legal issue match, factual similarity, statutory overlap, and citeability. 5 cases run in parallel.
④ Synthesis — The AI writes an executive memo based on the top 5 highest-scored cases.
